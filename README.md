## 💻 Scripts Included

### 🔹 Script 1 — System Identity Report
Displays system information such as:
- Kernel version
- Current user
- System uptime
- Date and time
- Linux distribution

---

### 🔹 Script 2 — FOSS Package Inspector
- Checks if Git is installed
- Displays package details
- Uses if-else and case statements

---

### 🔹 Script 3 — Disk and Permission Auditor
- Analyzes important system directories
- Displays permissions and disk usage
- Uses loops and system commands

---

### 🔹 Script 4 — Log File Analyzer
- Reads a log file
- Counts occurrences of a keyword (e.g., "error")
- Displays last matching lines

---

### 🔹 Script 5 — Open Source Manifesto Generator
- Takes user input
- Generates a personalized manifesto
- Saves output to a file
